How does Latitude change Weather?

This page contains an investigation of how cloudiness, humidity, wind, and temperature vary with Latitude. The data was collected from OpenWeatherMapAPI across a single day in July. Weather was collected for over 1000 cities. Cities were generated using a custom version of citipy, and could not be duplicates. I used a custom version of citipy to reduce bias towards coastal cities. All plots were made using Matplotlib and pandas.
